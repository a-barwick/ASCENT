# ASCENT

**Industrial operations beyond Earth.**

ASCENT is a persistent multiplayer economic simulation for the cislunar frontier.
Trade commodities, move freight, run production, and reconcile every committed
transaction against an authoritative ledger.

## Run

```sh
make bootstrap
make dev
```

Open `http://localhost:5173` and start a development session.

First playable MVP. Source-available under the ASCENT Proprietary Source License.
